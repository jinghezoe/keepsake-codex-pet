# Release checklist

Apply this checklist before any public repository, attachment, or shared archive.

- [ ] The represented person or appropriate rights holder approved the intended audience.
- [ ] Raw source photos and videos are absent unless separately and explicitly approved.
- [ ] No EXIF location, device, or timestamp metadata remains in published visuals.
- [ ] No private chats, email addresses, phone numbers, local paths, tokens, or account data appear.
- [ ] Rejected likeness attempts and debug folders are excluded.
- [ ] `pet.json` uses a relative `spritesheetPath` and declares `spriteVersionNumber: 2`.
- [ ] The atlas passes current v2 validation and final visual QA.
- [ ] Contact sheets and GIFs are readable but do not reveal unapproved private evidence.
- [ ] The skill/docs license and example-art license are stated separately.
- [ ] The README describes the pet as a memory or artwork, not as a replacement or spokesperson.
